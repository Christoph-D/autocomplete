# AI Autocomplete

A VS Code extension for inline autocomplete using any OpenAI compatible backend.

## Development

- **Build**: `pnpm run build`
- **Watch**: `pnpm run watch`
- **Lint**: `pnpm run lint`
- **Format** (must run after changes): `pnpm run format`

Always run `pnpm run format` and `pnpm run lint` after adding or changing anything.

## Testing

- Run all tests: `pnpm run test`

## Code Conventions

- TypeScript strict mode, ES2022 target
- ES modules with bundler resolution
- No comments unless requested
- Existing comments should remain in place unless they become incorrect
- Follow existing patterns in neighboring files
